Descubrimiento Automático de Permisos

Esta herramienta escanea automáticamente todas las rutas de Django en tu proyecto y crea permisos correspondientes. Es útil para generar permisos automáticamente basados en la estructura de URLs existente.

Proceso de Descubrimiento

1. Escaneo de URLs

Se escanean todas las aplicaciones Django y sus patrones de URL para identificar rutas únicas.

2. Filtrado y Categorización

Las rutas se categorizan automáticamente como vistas, menús o módulos basándose en patrones comunes.

3. Creación de Permisos

Se crean permisos únicos para cada ruta encontrada, evitando duplicados existentes.

4. Validación y Reporte

Se valida la integridad de los permisos creados y se genera un reporte detallado.
